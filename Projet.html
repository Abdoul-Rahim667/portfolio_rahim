Mes Projets en Informatique

Passionné par l’informatique, j’ai eu l’occasion de réaliser plusieurs projets personnels et académiques, principalement dans le développement web et la gestion de bases de données.

1. Application de Gestion des Vdp en programmation C

Objectif : Permettre au gouvernement de connaitre le nombre de vdp mais aussi d'attribuer un matricule a chaque vdp (ajout, supprimer, historique).

Technologies utilisées : Language C, MySQL

Compétences développées : Maitrise de la programmation C, interface graphique, logique métier.

2. Site Web pour un restaurant

Objectif : Créer un site pour promouvoir un restaurant local.

Technologies utilisées : HTML, CSS, un peu deJavaScript,

Compétences développées : HTML, CSS.

3. Portfolio Personnel

Objectif : Présenter mes compétences, passions et projets de façon professionnelle.

Technologies utilisées : HTML, CSS, un peu de JavaScript

Compétences développées : Présentation, organisation de contenu, esthétique web.

NB:Les projets présentés ont été réalisés dans le cadre de ma formation académique.
Ils ont été développés localement, principalement à des fins d’apprentissage et d’évaluation, ce qui explique qu’ils ne soient pas hébergés en ligne.
Toutefois, des démonstrations, captures d’écran ou fichiers sources peuvent être présentés en guise de preuve.
